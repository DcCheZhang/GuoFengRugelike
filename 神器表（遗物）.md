## 一、法宝分类说明

### 1.1 按功能分类

| 类型 | 说明 | 数量 |
|------|------|------|
| **进攻类** | 增加攻击、暴击、攻速等 | ~60个 |
| **防御类** | 增加防御、韧性、生命值等 | ~50个 |
| **功能类** | 提供特殊功能（复活、召唤等） | ~53个 |
| **特殊机制类** | 改变战斗机制 | ~50个 |

### 1.2 按品质分类

| 品质 | 说明 | 数量 |
|------|------|------|
| **普通（白）** | 基础法宝 | ~64个 |
| **稀少（绿）** | 强化法宝 | ~64个 |
| **传奇（紫）** | 稀有法宝 | ~64个 |
| **神话（橙）** | 限定法宝 | ~21个 |

### 1.3 获取途径

| 途径 | 说明 | 概率/条件 |
|------|------|-----------|
| **普通战斗掉落** | 战斗后概率获得 | 5%概率 |
| **精英战必掉** | 精英战斗后必定获得 | 100%概率 |
| **Boss战奖励** | Boss战后必定获得 | 100%概率 |
| **商店购买** | 商店中刷新并购买 | 随机刷新（3-5个） |
| **事件选择** | 事件中选择获得 | 取决于事件选项 |
| **章节结算奖励** | 通过章节后获得 | 100%概率 |
