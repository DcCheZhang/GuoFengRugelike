# UI面板与交互

> **文档版本**：v1.0  
> **文档定位**：面向策划与美术，详述所有面板布局与交互流程  
> **完成状态**：✅ 完整版（7个面板 + 交互流程）

---

## 一、主界面

### 1.1 布局结构

```
┌─────────────────────────────────────────────┐
│           Logo展示区（《万修列阵》）          │
│           版本号 v1.0                     │
├─────────────────────────────────────────────┤
│                                             │
│            [开始游戏]                        │
│            [继续游戏]（如有存档）             │
│            [设置]                           │
│            [退出]                           │
│                                             │
├─────────────────────────────────────────────┤
│          背景氛围元素（修仙场景）             │
│          制作组信息（点击查看）               │
└─────────────────────────────────────────────┘
```

### 1.2 信息层级

- **一级信息**：Logo、菜单项
- **二级信息**：版本号、制作组信息
- **三级信息**：背景氛围元素

### 1. 3 功能区划分

- **上方**：Logo展示区
- **中间**：菜单项（垂直排列）
- **下方**：背景氛围元素 + 制作组信息

### 1.4 视线重点

- **Logo**：最大，居中
- **开始游戏按钮**：次大，高亮

### 1.5 常驻信息 vs 触发显示信息

- **常驻信息**：Logo、菜单项、版本号
- **触发显示信息**：制作组信息（点击后弹出）

### 1.6 交互流程

| 操作 | 反馈 | 跳转 |
|------|------|------|
| **点击开始游戏** | 按钮高亮，播放"开始"音效 | 跳转至开局单位展示界面 |
| **点击继续游戏** | 按钮高亮，恢复存档 | 跳转至备战界面（存档位置） |
| **点击设置** | 弹出设置面板（音效/画质/语言） | 无跳转，面板内操作 |
| **点击退出** | 弹出确认对话框（"确定退出？"） | 确认后关闭游戏 |
| **点击制作组信息** | 弹出制作组信息面板 | 无跳转，面板内操作 |

| **悬停特殊能力** | 显示特殊能力详细说明 | 无跳转 |

## 三、备战界面（核心界面）

### 3.1 布局结构

```
┌─────────────────────────────────────────────┐
│ 关卡进度条：[当前大关]/[总大关] [Boss预警] │
├──────┬──────────────────────┬──────────────┤
│      │                      │  遗物展示区  │
│ 单位 │    待机展示区         │  （一列多行，│
│ 统计 │    （单位待机、闲逛、 │   超过自动   │
│ 面板 │     随机台词）        │   扩展多列） │
│      │                      │              │
│ 道具 │    [合成操作区域]     │              │
│ 栏   │                      │              │
├──────┴──────────────────────┴──────────────┤
│        战斗选择区（下一个可挑战的节点列表）     │
└─────────────────────────────────────────────┘
```

### 3.2 信息层级

- **一级信息**：待机展示区（氛围展示）、战斗选择区
- **二级信息**：单位统计面板、遗物展示区
- **三级信息**：道具栏、合成操作区域

### 3.3 功能区划分

- **左侧**：单位统计面板、道具栏
- **上方**：关卡进度条
- **中间**：待机展示区 + 合成操作区域
- **右侧**：遗物展示区
- **下方**：战斗选择区

### 3.4 视线重点

- **待机展示区**：最大，居中，动态展示
- **战斗选择区**：次大，清晰列表

### 3.5 常驻信息 vs 触发显示信息

- **常驻信息**：单位统计面板、关卡进度条、遗物展示区、战斗选择区
- **触发显示信息**：单位详情（悬停显示）、法宝详情（悬停显示）、合成预览（点击合成时显示）

### 3.6 交互流程

| 操作 | 反馈 | 跳转 |
|------|------|------|
| **点击战斗选择区节点** | 节点高亮，显示节点详情（类型/奖励/难度） | 点击确认后跳转至战斗界面 |
| **拖拽单位至合成区** | 单位图标跟随鼠标，合成区高亮 | 无跳转，显示合成预览 |
| **点击一键合成** | 播放"合成"音效，金光特效，属性对比动画 | 无跳转，刷新备战界面 |
| **点击遗物展示区法宝** | 显示法宝详情（名称/效果/品质） | 无跳转，可卸下法宝 |
| **拖拽法宝至单位** | 法宝图标跟随鼠标，单位高亮 | 无跳转，装备法宝 |
| **点击道具栏道具** | 显示道具详情，使用确认对话框 | 无跳转，使用道具 |
| **悬停单位** | 显示单位详情（属性/技能/法宝） | 无跳转 |

## 八、交互设计总结

### 8.1 完整流程图

```
启动 → 主界面 → [开始游戏] → 开局单位展示界面
  → [确认] → 备战界面
  → [选择战斗节点] → 战斗界面 → 自动战斗 → 战后结算界面
  → [获得奖励] → 备战界面（返回）
  → [进入商店] → 商店界面 → [离开商店] → 备战界面
  → [选择事件节点] → 事件界面 → [确认选项] → 备战界面
  → [Boss战胜利] → 章节结算 → 进入下一大关 / 返回主界面
```

### 8.2 操作方式定义

| 操作 | 功能 | 说明 |
|------|------|------|
| **点击** | 选择、确认、查看详情 | 单击选择，双击使用 |
| **拖拽** | 合成单位、装备法宝、出售物品 | 从单位栏拖到合成区/单位身上/出售面板 |
| **悬停** | 显示详情浮窗 | 悬停单位/法宝/道具显示详情 |
| **右键/长按** | 快速查看详情 | 替代悬停，显示更详细信息 |

### 8.3 状态与反馈

#### 按钮状态

- **正常**：高亮，可点击
- **悬停**：放大，颜色变化
- **按下**：缩小，颜色加深
- **禁用**：灰色，带原因提示（如"金币不足"）

#### 资源不足提示

- **金币不足**："囊中羞涩，先去赚点盘缠吧"
- **法宝栏已满**："法宝栏已满，先处理一下吧"

#### 复活中提示

- **灰色头像 + 倒计时**：如"剑修弟子（2回合后复活）"
- **文案提示**："还在回魂中..."

#### 特殊奖励提示

- **稀有单位/法宝获得**：强调动画（金光闪烁）+ 文案提示（"恭喜获得[名称]！"）

#### 合成成功反馈

- **视觉特效**：金光闪耀
- **音效提示**："叮——"（清脆音效）
- **属性变化展示**：显示合成前后的属性对比
