### 1.2 回合类型详解#

#### 普通战斗回合（14回合）
- **敌人数量**：回合1-5为5-8个，回合6-14为8-12个
- **敌人类型**：随机从当前章节敌人池中抽取
- **奖励**：金币50-100 + 随机单位/法宝（5%概率）

#### 遗物战斗回合（3回合）
- **敌人数量**：比普通战斗多2-3个
- **敌人类型**：随机从当前章节敌人池中抽取（强度略高）
- **奖励**：必得1个法宝（品质随机：普通60%/稀少30%/传奇10%）

#### 商店战斗回合（2回合）
- **敌人数量**：比普通战斗多1-2个
- **敌人类型**：随机从当前章节敌人池中抽取
- **奖励**：金币100-150 + 战后进入商店

#### Boss战回合（第20回合）
- **敌人数量**：1个Boss + 小兵10-15个
- **敌人类型**：章节Boss + 随机小兵
- **奖励**：大量金币（300-500）+ 特殊奖励（章节解锁/新单位/新法宝）

## 三、循环规则（下一循环生成规则）#

### 3.1 第一循环（第1-20回合）#

- **敌人池**：当前章节敌人池（如第一章：凡间敌人池）
- **难度**：标准（×1.0）
- **金币奖励**：标准（50-300金币）

### 3.2 第二循环（第21-40回合）#

**触发条件**：玩完第20回合（Boss战胜利）

**生成规则**：
1. **敌人池扩展**：加入下一章节的敌人（如第二章：幽冥敌人池）
2. **难度提升**：敌人属性 × 1.2（相对第一循环）
3. **金币奖励提升**：金币奖励 × 1.2
4. **新遗物**：扩展遗物池（加入下一章节的遗物）

**具体规则**：
- **敌人类型**：从扩展池中随机抽取（50%当前章节 + 50%下一章节）
- **敌人数量**：比第一循环多2-3个
- **Boss战**：使用下一章节Boss

### 3.3 第三循环及以后（第41回合+）#

**生成规则**：
1. **敌人池继续扩展**：每循环加入新章节的敌人
2. **难度继续提升**：敌人属性每循环 × 1.2（累积）
3. **金币奖励继续提升**：金币奖励每循环 × 1.2（累积）
4. **遗物池继续扩展**：每循环加入新章节的遗物

**循环上限**：
- **最多9个循环**（对应9大关）
- 玩完9个循环后，进入"混沌模式"（随机机制，无限循环）

## 五、敌人生成算法#

### 5.1 普通战斗敌人生成#

```
函数：生成普通战斗敌人(回合号，章节号)
{
  // 1. 确定敌人数量
  基础数量 = 5 + floor(回合号 / 5); // 每5回合增加1个敌人
  随机增减 = rand(-1, 2); // -1到2的随机整数
  敌人数量 = 基础数量 + 随机增减;
  
  // 2. 从敌人池中随机选择敌人类型
  敌人池 = 当前章节敌人池 + 已解锁章节敌人池;
  敌人类型列表 = 随机选择(敌人池, 敌人数量);
  
  // 3. 应用难度倍数
  难度倍数 = 循环难度倍数 × 循环内难度倍数;
  for (每个敌人 in 敌人类型列表) {
    应用难度倍数(敌人, 难度倍数);
  }
  
  return 敌人类型列表;
}
```

### 5.2 Boss战敌人生成#

```
函数：生成Boss战敌人(章节号)
{
  // 1. Boss
  Boss = 当前章节Boss;
  应用难度倍数(Boss, 难度倍数 × 1.5); // Boss难度翻倍
  
  // 2. 小兵
  小兵数量 = 10 + floor(章节号 / 3); // 每3章增加1个小兵
  小兵列表 = 生成普通战斗敌人(19, 章节号); // 复用普通战斗生成逻辑
  
  return [Boss] + 小兵列表;
}
```

## 六、奖励生成算法#

### 6.1 金币奖励#

```
函数：计算金币奖励(敌人总属性, 回合类型)
{
  基础金币 = 敌人总属性 / 10; // 敌人越强，金币越多
  
  // 回合类型加成
  if (回合类型 == "Boss战") {
    类型加成 = 3.0; // Boss战金币×3
  } else if (回合类型 == "商店战斗") {
    类型加成 = 2.0; // 商店战斗金币×2
  } else {
    类型加成 = 1.0;
  }
  
  金币奖励 = 基础金币 × 类型加成 × 金币奖励倍数;
  return 金币奖励;
}
```

### 6.2 遗物奖励#

```
函数：生成遗物奖励(回合号，章节号)
{
  // 1. 确定品质
  随机值 = rand(0, 1);
  if (随机值 < 0.6) {
    品质 = "普通";
  } else if (随机值 < 0.9) {
    品质 = "稀少";
  } else {
    品质 = "传奇";
  }
  
  // 2. 从遗物池中随机选择
  遗物池 = 当前章节遗物池 + 已解锁章节遗物池;
  遗物列表 = 按品质过滤(遗物池, 品质);
  遗物 = 随机选择(遗物列表, 1);
  
  return 遗物;
}
```

## 八、程序实现建议#

### 8.1 回合生成器（TypeScript）#

```typescript
class RoundGenerator {
  // 生成指定回合的敌人
  generateEnemies(round: number, chapter: number): EnemyUnit[] {
    const cycle = Math.floor((round - 1) / 20) + 1; // 当前循环（1-9）
    const roundInCycle = (round - 1) % 20 + 1; // 循环内回合数（1-20）
    
    // 1. 确定敌人数量
    const baseCount = 5 + Math.floor(roundInCycle / 5);
    const random0ffset = Math.floor(Math.random() * 4) - 1; // -1 to 2
    const enemyCount = baseCount + random0ffset;
    
    // 2. 从敌人池中随机选择
    const enemyPool = this.getEnemyPool(chapter, cycle);
    const selectedEnemies = this.randomSelect(enemyPool, enemyCount);
    
    // 3. 应用难度倍数
    const difficultyMultiplier = this.getDifficultyMultiplier(cycle, roundInCycle);
    return selectedEnemies.map(enemy => this.applyDifficulty(enemy, difficultyMultiplier));
  }
  
  // 获取敌人池（当前章节 + 已解锁章节）
  private getEnemyPool(currentChapter: number, cycle: number): EnemyType[] {
    let pool: EnemyType[] = [];
    for (let i = 1; i <= currentChapter; i++) {
      pool = pool.concat(CHAPTER_ENEMY_PO0LS[i]);
    }
    return pool;
  }
  
  // 获取难度倍数
  private getDifficultyMultiplier(cycle: number, roundInCycle: number): number {
    const cycleMultiplier = Math.pow(1.2, cycle - 1); // 循环间倍数
    let roundMultiplier = 1.0;
    if (roundInCycle <= 5) roundMultiplier = 1.0 + (roundInCycle - 1) * 0.02;
    else if (roundInCycle <= 10) roundMultiplier = 1.1 + (roundInCycle - 6) * 0.02;
    else if (roundInCycle <= 15) roundMultiplier = 1.2 + (roundInCycle - 11) * 0.04;
    else if (roundInCycle <= 19) roundMultiplier = 1.4 + (roundInCycle - 16) * 0.05;
    else roundMultiplier = 1.5; // Boss战
    return cycleMultiplier * roundMultiplier;
  }
}
```
