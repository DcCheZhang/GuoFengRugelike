## 一、属性接口说明

### 1.1 统一属性枚举（我方与敌方共用）

| 属性 | 类型 | 说明 | 取值范围 |
|------|------|------|-----------|
| **HP** | number | 生命值（当前） | 1-99999 |
| **maxHp** | number | 生命值（最大） | 1-99999 |
| **attack** | number | 攻击力 | 1-9999 |
| **defense** | number | 防御力 | 1-9999 |
| **attackSpeed** | number | 攻击间隔（秒） | 0.1-10.0 |
| **moveSpeed** | number | 移动速度（像素/秒） | 50-500 |
| **range** | number | 攻击距离（像素，0=近战） | 0-500 |
| **critRate** | number | 暴击率（0-1） | 0-1.0 |
| **critDamage** | number | 暴击伤害倍数 | 1.0-10.0 |
| **dodgeRate** | number | 闪避率（0-1） | 0-1.0 |
| **toughness** | number | 韧性（减少被暴击率） | 0-100 |

### 1.2 等阶倍数

| 等阶 | 倍数 | 说明 |
|------|------|------|
| **普通（白）** | × 1.0 | 基础值 |
| **稀少（绿）** | × 2.0 | 强化版 |
| **传奇（紫）** | × 4.0 | 稀有版 |
| **神话（橙）** | × 8.0 | 限定版（限1个） |

## 四、数值平衡说明

### 4.1 我方单位 vs 敌方单位

**普通战斗（回合1-14）**：
- 我方普通单位（白）：HP 100-200, 攻击 10-20
- 敌方普通杂兵：HP 50-150, 攻击 8-15
- **结论**：我方单位略强，但数量劣势时需要策略

**精英战**：
- 我方稀少单位（绿）：HP 200-300, 攻击 20-40
- 敌方精英怪：HP 200-400, 攻击 20-40
- **结论**：势均力敌，需要合理搭配

**Boss战（回合20）**：
- 我方传奇/神话单位：HP 400-1600, 攻击 40-160
- 敌方Boss：HP 500-1000, 攻击 40-80
- **结论**：Boss血量高，需要持续输出

### 4.2 数值调整建议

- **如果太简单**：提高敌方攻击力10-20%
- **如果太难**：降低敌方防御力10-15%
- **如果战斗时间太长**：提高我方攻击力或降低敌方HP
