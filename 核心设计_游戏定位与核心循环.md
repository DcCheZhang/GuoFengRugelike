## 一、游戏定位

### 1.1 基本信息

| 项目 | 内容 |
|------|------|
| **游戏名称** | 暂定《万修列阵》或《修仙草台班子》 |
| **游戏类型** | Steam 单机 Roguelike + 自动战斗 + 合成扩军 + 随机开局 |
| **题材** | 国风神怪 / 草台班子伐天 |
| **平台** | Steam 单机（明确不含手游商业化设计） |
| **目标玩家** | 喜欢自走棋、Roguelike、国风玄幻题材的PC玩家；偏好策略构筑但厌倦手操的玩家 |
| **核心卖点** | • 国风草台风幽默 + 修仙世界观<br>• 自动战斗 + 深度构筑空间<br>• 合成扩军带来爽感与策略性<br>• 随机开局带来差异化体验 |
| **体验关键词** | 修仙、草台班子、自动战斗、合成扩军、构筑 |
| **参考作品** | 《How Many Dudes Demo》（核心玩法参考）<br>《吸血鬼幸存者》（自动战斗爽感）<br>《黑帝斯》（Roguelike节奏）<br>《自走棋》（阵容构筑） |

### 1.2 四个为什么

1. **为什么这一局好玩？** → 随机开局单位 + 随机单位/法宝组合，每局构筑不同
2. **为什么阵容构筑有变化？** → 四阶单位 + 213个法宝，流派多样
3. **为什么玩家愿意反复开局？** → Roguelike死亡重置 + 局外解锁 + 成就收集
4. **为什么这个项目做得出来？** → 中小团队可行，美术可复用，程序框架清晰

## 三、核心玩法概述

玩家开局**随机获得一名修仙者/妖怪/神兵**，带领一支草台班子，通过**自动战斗**击败一波波敌人，战后**招募新单位、获取法宝、合成升级**，不断扩军强化，最终挑战**天庭 Boss**，成功则进入下一章，失败则重整旗鼓再开局。

## 四、核心循环与单局流程

### 4.1 流程图

```
随机获得开局单位 → 备战界面（合成/调整/装备法宝）
     ↓
选择战斗节点 → 自动战斗（双方从两侧冲锋交战） → 战后结算
     ↓
招募新单位（3选1） / 获取法宝 / 获得金币
     ↓
返回备战界面（合成升级 / 调整阵容 / 装备法宝）
     ↓
选择下一战斗节点 → ... → Boss战 → 章节结算 → 进入下一章 / 失败重修
```

### 4.2 关键标注

| 要素 | 说明 |
|------|------|
| **玩家决策点** | 单位招募（3选1）、法宝选择、合成决策、阵容调整 |
| **随机性环节** | 开局单位、单位池、法宝池、商店商品、事件结果 |
| **不可逆决策** | 单位合成（消耗材料）、金币花费 |
| **单局时长** | 目标 30-60 分钟（含多章节） |

### 4.3 单局流程详解

#### 阶段1：开局
- 系统随机生成1个单位（等阶随机：普通60%/稀少30%/传奇10%）
- 玩家可花费100灵石重新随机（每局限3次）
- 确认后进入备战界面

#### 阶段2：备战
- **合成操作**：3个同阶同级 → 升1阶；5个同阶同级 → 升2阶
- **阵容调整**：拖拽调整单位站位
- **装备法宝**：为生还单位装备法宝（右侧法宝栏）
- **使用道具**：使用回复类/增益类道具

#### 阶段3：选择战斗节点
- 每章提供2-3条路径，难度和奖励不同
- 节点类型：普通战斗 / 遗物战斗 / 商店战斗 / 精英战 / Boss战 / 事件节点

#### 阶段4：自动战斗
- 双方从两侧冲锋交战
- 单位自动攻击范围内敌人，技能自动释放
- 玩家可调整倍速（0.5x / 1.0x / 1.5x）
- 玩家可使用道具（战斗中）

#### 阶段5：战后结算
- 显示战斗结果（胜利/失败）
- 单位表现统计（伤害排名、承伤排名、治疗排名、存活回合）
- MVP评选与称号展示
- 获得奖励：金币 + 新单位（3选1）+ 法宝 + 道具

#### 阶段6：循环
- 返回备战界面，继续强化阵容
- 选择下一战斗节点，进入下一轮战斗
- 击败Boss后，章节结算，进入下一章

## 五、游戏核心数据

### 5.1 单位数据

| 类别 | 数量 | 说明 |
|------|------|------|
| **我方单位** | 27种 | 4个等阶：普通（白）/ 稀少（绿）/ 传奇（紫）/ 神话（橙，限1） |
| **敌人** | 35种 | 普通杂兵 / 精英怪 / Boss |
| **法宝（遗物）** | 213个 | 4个品质：普通（白）/ 稀少（绿）/ 传奇（紫）/ 神话（橙） |

### 5.2 关卡数据

| 项目 | 数据 |
|------|------|
| **章节结构** | 9大关 × 20小关 = 180关 |
| **小关结构** | 14普通战斗 + 2商店 + 3遗物战斗 + 1Boss战 = 20回合 |
| **单局时长** | 目标 30-60 分钟 |

### 5.3 数值框架

| 属性 | 普通 | 稀少 | 传奇 | 神话 |
|------|------|------|------|------|
| **HP（生命值）** | 100-200 | 200-400 | 400-800 | 800-1500 |
| **攻击** | 10-20 | 20-40 | 40-80 | 80-150 |
| **防御** | 5-10 | 10-20 | 20-40 | 40-80 |
| **攻速** | 1.0-1.5秒 | 0.8-1.2秒 | 0.6-1.0秒 | 0.4-0.8秒 |
| **移速** | 100 | 110 | 120 | 130 |

**伤害公式**：
```
伤害 = (攻击 - 防御) × 技能系数 × 随机波动(0.9-1.1) × 暴击倍数（如暴击则 ×2）
```
