## 一、开局系统

### 1.1 设计目标

- 提供**随机开局体验**（每局随机获得1个单位）
- 简化开局流程，让玩家快速进入核心玩法
- 随机单位不影响长期策略，保持构筑空间

### 1.2 开局单位生成规则

- **随机选择**：从已解锁的单位池中随机选择1个单位
- **等级设定**：开局单位固定为1阶
- **等阶设定**：等阶随机（普通60%、稀少30%、传奇10%）
- **属性调整**：开局单位属性为标准的80%（鼓励早期替换）

### 1.3 开局流程

1. 点击"开始游戏"
2. 系统随机生成1个开局单位
3. 展示开局单位信息（名称、等阶、属性）
4. 玩家确认后进入备战界面

### 1.4 开局单位展示界面

```
┌─────────────────────────────────────────────┐
│              随机开局单位                   │
├─────────────────────────────────────────────┤
│                                             │
│           [单位头像]                        │
│                                             │
│   名称：剑修弟子                            │
│   等阶：普通（白）                          │
│   类型：战士流                              │
│   属性：攻击15 / 防御8 / 生命120           │
│                                             │
│   特殊能力：剑气斩（直线伤害）               │
│                                             │
├─────────────────────────────────────────────┤
│            [确认]  [重新随机]                │
└─────────────────────────────────────────────┘
```

### 1.5 重新随机机制

- 玩家可点击"重新随机"按钮，消耗100灵石重新随机选择开局单位
- 每局游戏限用3次重新随机机会

## 二、单位系统

### 2.1 设计目标

- **四等阶**：普通（白）/ 稀少（绿）/ 传奇（紫）/ 神话（橙，限1个）
- 单位可**无限获得**，不同单位代表不同流派
- 部分特殊单位拥有**简单AI**（走位、躲避、绕后）

### 2.2 单位基础属性框架

| 属性 | 说明 | 普通 | 稀少 | 传奇 | 神话 |
|------|------|------|------|------|------|
| **HP（生命值）** | 生存能力 | 100-200 | 200-400 | 400-800 | 800-1500 |
| **攻击** | 伤害输出 | 10-20 | 20-40 | 40-80 | 80-150 |
| **防御** | 伤害减免 | 5-10 | 10-20 | 20-40 | 40-80 |
| **攻速** | 攻击间隔（秒） | 1.0-1.5 | 0.8-1.2 | 0.6-1.0 | 0.4-0.8 |
| **移速** | 移动速度 | 100 | 110 | 120 | 130 |
| **射程** | 攻击距离 | 近战/远程 | 近战/远程 | 近战/远程 | 近战/远程 |

### 2.3 单位设计示例（10名，覆盖不同流派）

- **背景**："刚入门的修仙者，火球术还没练熟"

**4. 石灵（稀少 / 坦克流）**

**5. 雷修（稀少 / 法师流）**

## 三、合成系统

### 3.1 设计目标

- 明确合成规则，提供**数值成长爽感**
- 合成后的**属性成长公式**清晰
- 合成**上限和边界条件**明确

### 3.2 合成规则

- **3个同阶同级 → 升1阶**（如：3个1阶普通 → 1个2阶普通）
- **5个同阶同级 → 升2阶**（如：5个1阶普通 → 1个3阶普通）
- **合成后属性成长公式**：`新属性 = 原属性 × (1 + 0.5 × 新阶数)`
- **合成上限**：每单位最高5阶
- **边界条件**：神话单位无法合成（限1个）

### 3.3 合成操作界面交互

- **拖拽合成**：从单位栏拖拽3个（或5个）同阶同级单位到合成区
- **一键合成**：点击"一键合成"按钮，自动合成所有可合成单位
- **合成预览**：合成前显示合成后的属性预览

## 四、遗物系统（法宝系统）

### 4.1 设计目标

- 遗物分类清晰，形成**不同流派**
- 遗物获取途径多样
- 遗物展示规则明确

### 4.2 遗物分类

| 类型 | 说明 | 示例 |
|------|------|------|
| **进攻类** | 增加攻击、暴击、攻速等 | 斩仙飞刀、番天印 |
| **防御类** | 增加防御、韧性、生命值等 | 八卦紫绶衣、金刚圈 |
| **功能类** | 提供特殊功能（复活、召唤等） | 招妖幡、替身傀儡 |
| **特殊机制类** | 改变战斗机制 | 诛仙阵图、周天星斗大阵 |

### 4.3 遗物获取途径

- **战斗掉落**：普通战斗概率掉落（5%），精英战斗必掉（100%）
- **商店购买**：商店中随机刷新（3-5个）
- **事件获得**：事件中选择获得

### 4.4 遗物构筑差异设计

- **狂战士流**：选择增加攻击、暴击、攻速的法宝
- **坦克流**：选择增加防御、韧性、生命值的法宝
- **法师流**：选择增加法术伤害、法力值的法宝
- **召唤流**：选择增加召唤物数量、强度的法宝
- **辅助流**：选择增加治疗量、增益效果的法宝

### 4.5 遗物展示规则

- 备战界面**右侧**展示已装备的法宝
- **一列多行**，超过自动扩展为多列
- 悬停显示法宝详情（名称、效果、品质）

### 4.6 遗物设计示例（8个）

**1. 斩仙飞刀（进攻类 / 橙色 / 神话）**
- **效果**：攻击时10%概率"斩仙"（一击必杀，对Boss无效）
- **获取途径**：Boss战奖励
- **背景**："陆压道人的法宝，专斩仙神"

**2. 番天印（进攻类 / 紫色 / 传奇）**
- **效果**：攻击时20%概率造成范围伤害（相当于攻击力150%）
- **获取途径**：精英战斗掉落
- **背景**："广成子的法宝，番天覆地"

**3. 八卦紫绶衣（防御类 / 紫色 / 传奇）**
- **效果**：减少受到伤害20%，增加韧性15%
- **获取途径**：商店购买
- **背景**："太上老君的护体法宝，八卦护身"

**4. 金刚圈（防御类 / 蓝色 / 稀少）**
- **效果**：增加防御30%，但攻速-10%
- **获取途径**：普通战斗掉落
- **背景**："金刚不坏之身的象征"

**5. 招妖幡（功能类 / 橙色 / 神话）**
- **效果**：每场战斗开始时有30%概率召唤1个随机妖怪（2阶）
- **获取途径**：事件选择
- **背景**："女娲的法宝，能召唤天下妖怪"

**6. 替身傀儡（功能类 / 紫色 / 传奇）**
- **效果**：死亡时自动复活，恢复50%生命值（每场战斗限1次）
- **获取途径**：商店购买
- **背景**："茅山术的替身傀儡术"

**7. 诛仙阵图（特殊机制类 / 橙色 / 神话）**
- **效果**：战斗开始时展开"诛仙阵"，范围内敌人每秒受到法术伤害（相当于攻击力50%）
- **获取途径**：隐藏Boss掉落
- **背景**："通天教主的法宝，诛仙灭神"

**8. 周天星斗大阵（特殊机制类 / 紫色 / 传奇）**
- **效果**：每有5个同族单位，全体增加攻击15%、防御15%
- **获取途径**：章节结算奖励
- **背景**："妖族的护族大阵，周天星斗加持"

## 五、道具系统

### 5.1 设计目标

- 道具存放于备战界面，可**点击使用**
- 道具分类清晰，使用限制明确

### 5.2 道具分类

| 类型 | 说明 | 示例 |
|------|------|------|
| **回复类** | 恢复生命值、复活单位 | 九转金丹、还魂香 |
| **增益类** | 临时增加属性 | 狂暴丹、御风符 |
| **控制类** | 控制敌人 | 定身符、捆仙绳 |
| **特殊类** | 特殊效果 | 传送符、隐身符 |

### 5.3 道具使用限制

- **战斗前使用**：如增益类道具（持续整场战斗）
- **战斗中使用**：如回复类道具（即时生效）
- **随时使用**：如特殊类道具（无限制）

### 5.4 道具设计示例（6个）

**1. 九转金丹（回复类）**
- **效果**：恢复全体单位50%生命值
- **使用限制**：战斗中可使用1次
- **获取途径**：商店购买 / 事件获得

**2. 还魂香（回复类）**
- **效果**：复活1名阵亡单位，恢复30%生命值
- **使用限制**：战斗中可使用1次
- **获取途径**：精英战斗掉落

**3. 狂暴丹（增益类）**
- **效果**：全体攻击 +30%，攻速 +20%，持续30秒
- **使用限制**：战斗前使用，持续整场战斗
- **获取途径**：商店购买

**4. 御风符（增益类）**
- **效果**：全体移速 +30%，闪避 +20%，持续30秒
- **使用限制**：战斗前使用，持续整场战斗
- **获取途径**：普通战斗掉落

**5. 定身符（控制类）**
- **效果**：定身范围内所有敌人5秒
- **使用限制**：战斗中可使用1次
- **获取途径**：商店购买

**6. 捆仙绳（控制类）**
- **效果**：捆绑1个敌人，使其无法行动10秒
- **使用限制**：战斗中可使用1次
- **获取途径**：精英战斗掉落

## 六、关卡系统

### 6.1 设计目标

- 节点类型多样，提供**不同挑战**
- 关卡进度结构清晰
- 难度曲线合理

### 6.2 节点类型

| 类型 | 说明 | 奖励 |
|------|------|------|
| **普通战斗** | 标准战斗，难度较低 | 金币 + 随机单位/法宝 |
| **遗物战斗** | 战斗后必得法宝 | 战斗难度略高 |
| **商店战斗** | 战斗后进入商店 | 战斗难度中等 |
| **精英战** | 高强度战斗，必得稀有以上奖励 | 稀有单位/法宝 |
| **Boss战** | 章节末尾高强度战斗 | 大量资源 + 特殊奖励 |
| **事件节点** | 触发事件，选择结果 | 风险与收益对等 |

### 6.3 关卡进度结构

- **九重天**：9大关 × 20小关 = 180关
- 每关19小关 + 1个Boss战
- 每3小关为一个节点组，提供选择空间

### 6.4 节点选择机制

- **分支路径**：每章提供2-3条路径，难度和奖励不同
- **线性推进**：部分章节强制线性推进（如Boss战前）

### 6.5 难度曲线设计

- **前期（1-3关）**：教学 + 爽感，低压力
- **中期（4-15关）**：构筑成型，压力渐增
- **后期（16-19关）**：极限考验，Boss战
- **跨章节难度跃升**：新机制引入，数值台阶

### 6.6 特殊机制关卡

- **限制单位数量**：如"仅限5个单位参战"
- **仅限某流派**：如"仅限法师流单位"
- **天劫降临**：如"每10波降下天劫，对全场造成伤害"

## 七、战斗系统

### 7.1 设计目标

- **自动战斗核心规则**清晰
- 战斗时长控制合理
- 伤害计算公式简单清晰

### 7.2 自动战斗核心规则

- 双方从**两侧冲锋交战**
- 单位**自动攻击**范围内敌人
- 技能**自动释放**（冷却时间到即释放）
- 玩家可**调整站位**（备战界面）和**使用道具**（战斗中）

### 7.3 战斗时长控制目标

- **普通战斗**：目标 1-2 分钟
- **精英战**：目标 2-3 分钟
- **Boss战**：目标 3-5 分钟

### 7.4 伤害计算公式框架

```
伤害 = (攻击 - 防御) × 技能系数 × 随机波动(0.9-1.1) × 暴击倍数（如暴击则 ×2）
```

### 7.5 特殊战斗机制

- **地形效果**：如"沼泽地"（移速-50%）、"火山"（每秒受到法术伤害）
- **天气效果**：如"暴雨"（火系伤害-50%）、"暴雪"（移速-30%）

### 7.6 战斗失败/逃跑机制

- **战斗失败**：单位全灭则失败，可选择"重修"（消耗重修令）或"返回主界面"
- **逃跑**：战斗中可逃跑（消耗逃跑符），但视为失败

## 八、商店系统

### 8.1 设计目标

- 战后进入商店，**提供强化机会**
- 商品类型多样，价格体系合理

### 8.2 商品类型

| 类型 | 说明 |
|------|------|
| **单位** | 随机刷新1-3个单位（普通/稀少/传奇） |
| **遗物（法宝）** | 随机刷新1-2个法宝（稀少/传奇） |
| **道具** | 随机刷新2-3个道具 |
| **刷新机会** | 花费金币刷新商品 |

### 8.3 价格体系

- **单位**：普通（100金币）/ 稀少（300金币）/ 传奇（1000金币）
- **法宝**：稀少（500金币）/ 传奇（1500金币）/ 神话（5000金币）
- **道具**：回复类（200金币）/ 增益类（300金币）/ 控制类（400金币）

### 8.4 通货膨胀控制

- 每章物价上涨10%（如第二章单位价格 ×1.1）
- 金币获取量随章节增加而提升

### 8.5 出售/回购规则

- **出售单位**：返还50%购买价格
- **出售法宝**：返还50%购买价格
- **回购**：商店中可回购出售的物品（返还100%出售价格）

## 九、死亡与复活系统

### 9.1 设计目标

- 单位死亡后有**惩罚与复活机会**
- 避免单位永久死亡，保持游戏体验

### 9.2 单位死亡

- 进入**复活倒计时**（默认4回合）
- 倒计时期间**无法参战**
- 复活后恢复50%生命值

### 9.3 复活倒计时可视化

- 备战界面中显示**灰色头像 + 倒计时**
- 文案提示："还在回魂中..."

### 9.4 可加速/减少复活时间的机制

- **道具**：如"还魂香"（立即复活）
- **法宝**：如"替身傀儡"（死亡时自动复活）
- **事件**：如"仙人指路"（减少复活时间2回合）

## 十、章节推进与局外成长

### 10.1 设计目标

- 章节结构清晰，提供**长期目标**
- 局外解锁内容多样，增加**重复可玩性**

### 10.2 章节结构

| 章节 | 名称 | 主题 |
|------|------|------|
| **第一章** | 凡间 | 修仙入门，草台班子组建 |
| **第二章** | 幽冥 | 阴间探险，鬼修势力 |
| **第三章** | 妖界 | 妖族领地，万妖云集 |
| **第四章** | 龙宫 | 海底龙宫，龙族势力 |
| **第五章** | 天庭外围 | 天兵天将，初战天庭 |
| **第六章** | 蟠桃园 | 蟠桃盛会，暗流涌动 |
| **第七章** | 南天门 | 天庭要塞，恶战连连 |
| **第八章** | 凌霄殿 | 玉帝面前，最终对决 |
| **第九章** | 混沌 | 天道之外，终极秘密 |

### 10.3 局外解锁内容

| 类型 | 解锁条件 | 说明 |
|------|----------|------|
| **新单位** | 累积局数 | 如"累积通关5局解锁龙族后裔" |
| **新法宝** | 达成特定成就 | 如"无伤通关第九章解锁斩仙飞刀" |
| **新事件** | 通关某章节 | 如"通关第三章解锁妖界事件池" |
| **新难度** | 通关某难度 | 如"通关普通难度解锁困难难度" |

### 10.4 进度保存机制

- **Roguelike传统**：单局结束才保存进度
- **章节通关进度**：通关某章节后永久解锁该章节
- **局外解锁进度**：永久保存

### 10.5 难度层级

| 难度 | 说明 |
|------|------|
| **普通** | 标准难度，适合新手 |
| **困难** | 敌人属性 +20%，适合老手 |
| **逆天** | 敌人属性 +50%，适合硬核玩家 |
| **混沌** | 敌人属性 +100%，随机机制，适合极限挑战 |
